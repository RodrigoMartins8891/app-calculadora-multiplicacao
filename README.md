📱 App Calculadora de Multiplicação

Aplicativo simples desenvolvido com React Native que realiza a multiplicação entre dois números informados pelo usuário e exibe o resultado na tela.

Este projeto foi criado com o objetivo de praticar conceitos básicos de desenvolvimento mobile, como criação de componentes, manipulação de estado e entrada de dados.

🚀 Tecnologias utilizadas

React Native

JavaScript

Expo (se você usou)

Componentes funcionais

useState

📸 Funcionalidades

Inserir dois números

Calcular a multiplicação

Exibir o resultado na tela

Interface simples e intuitiva

⚙️ Como executar o projeto

1️⃣ Clone o repositório

git clone (https://github.com/RodrigoMartins8891/app-calculadora-multiplicacao)

2️⃣ Entre na pasta do projeto

cd app-calculadora-multiplicacao

3️⃣ Instale as dependências

npm install

4️⃣ Execute o projeto

npx expo start

Depois disso, abra no Expo Go ou em um emulador Android/iOS.
